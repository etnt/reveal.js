GIT intro

Create a Git repo


$ mkdir hello
$ cd hello
$ git init

Add a file to be version controlled


$ echo "Hello" > README
$ git add README
$ git commit -am "Adding README"

Ignore the "-a" switch for now!

Edit some more...


$ echo "World" >> README

...check what has changed...


$ git diff
diff --git a/README b/README
index e965047..f9264f7 100644
--- a/README
+++ b/README
@@ -1 +1,2 @@
 Hello
+World

...and commit it!


$ git commit -am "Modified README"
